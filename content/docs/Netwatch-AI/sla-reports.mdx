## How to Create an SLA Report in Netwatch AI

Follow these steps to configure and automate SLA reports using Netwatch AI.

### Step 1: Access the Netwatch AI Bot

Click on the Bot icon located at the bottom right of the interface to open the Netwatch AI Bot.

<ZoomableImage src="/images/ai.png" alt="Step 2: Create a New Dashboard" />

Once the bot window opens, navigate to the **Reports** tab. This will take you to the Report Configurations page, where you can manage automated SLA report configurations with custom schedules and settings.

<ZoomableImage src="/images/ai1.png" alt="Step 2: Create a New Dashboard" />
---

### Step 2: Create Your First Configuration

Click the **"Create Your First Configuration"** button to begin setting up a new automated report.

<ZoomableImage src="/images/ai2.png" alt="Step 2: Create a New Dashboard" />
---

### Step 3: Configure Report Settings

Fill in the configuration form with your desired settings:

- **Configuration Name:** Enter a descriptive name (e.g., *Windows Server Monthly SLA Report*)
- **OS Type:** Select the operating system (e.g., *Windows*)
- **Report Period:** Choose the time range (e.g., *Last 30 Days*)
- **Frequency:** Set how often the report runs (e.g., *Monthly*)
- **Host Groups:** Select which host groups to include
- **Scheduled Time:** Set the time of day (e.g., *09:00 AM*)
- **Day of Month:** Choose which day (e.g., *30*)
- **Email Recipients:** Add comma-separated email addresses
- **Configuration is active:** Ensure this option is checked

<ZoomableImage src="/images/ai4.png" alt="Step 2: Create a New Dashboard" />
---

### Step 4: Send or Schedule the Report

Once created, your configuration will appear in the list. You can:

- Click **Send** to manually trigger the report immediately
- Click **Edit** to modify settings
- Click **Remove** to delete the configuration

The report will run automatically based on your specified schedule.

<ZoomableImage src="/images/ai3.png" alt="Step 2: Create a New Dashboard" />
